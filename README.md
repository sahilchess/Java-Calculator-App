# Java Calculator App

A simple command-line calculator built in Java for practicing programming fundamentals.

## Features
- Supports basic arithmetic operations: `+`, `-`, `*`, `/`
- Handles invalid number input with friendly error messages
- Prevents division by zero
- Allows exiting at any prompt by typing `exit`

## Requirements
- Java JDK 8 or later

## How to Run
1. Compile the program:
   ```bash
   javac Calculator.java
   ```
2. Run the calculator:
   ```bash
   java Calculator
   ```

## Example
```text
Simple Calculator
Type 'exit' to quit

Enter first number: 12
Enter operator (+, -, *, /): *
Enter second number: 3
Result: 36.0
```

## Project Structure
- `Calculator.java` - main calculator source code

## Notes
This is a beginner-friendly project focused on learning Java input handling, conditionals, and loops.
